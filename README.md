# VideoCache


![image](https://github.com/xihuan22d/VideoCache/blob/master/screenshot/1.png)


androidStudio Gradle导入:

先添加jitpack仓库:

allprojects {

    repositories {  
	
        maven { url 'https://www.jitpack.io' }
		
    }
	
}

接着导入项目依赖库即可:

dependencies {

    implementation 'com.github.xihuan22d:VideoCache:1.0.0'
	
}
